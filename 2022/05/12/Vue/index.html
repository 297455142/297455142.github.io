5.2 Depдҫқиө–ж”¶йӣҶ

е®ғз”ЁжқҘж”¶йӣҶдҫқиө–гҖҒеҲ йҷӨдҫқиө–е’Ңеҗ‘дҫқиө–еҸ‘йҖҒж¶ҲжҒҜзӯүгҖӮ
дҫқиө–ж”¶йӣҶиҝҮзЁӢпјҡи°Ғз”ЁеҲ°дәҶиҝҷдёӘж•°жҚ®з§°дёәвҖңи°Ғдҫқиө–дәҶиҝҷдёӘж•°жҚ®вҖқпјҢз»ҷжҜҸдёӘж•°жҚ®йғҪе»әз«ӢдёҖдёӘдҫқиө–ж•°з»„пјҲеӣ дёәдёҖдёӘж•°жҚ®еҸҜиғҪиў«еӨҡеӨ„дҪҝз”ЁпјүпјҢи°Ғдҫқиө–дәҶиҝҷдёӘж•°жҚ®жҲ‘д»¬е°ұжҠҠи°Ғж”ҫе…ҘеҲ°иҝҷдёӘдҫқиө–ж•°з»„дёӯпјҢеҪ“иҝҷдёӘж•°жҚ®еҸ‘з”ҹеҸҳеҢ–зҡ„ж—¶еҖҷпјҢжҲ‘д»¬е°ұеҺ»е®ғеҜ№еә”зҡ„дҫқиө–ж•°з»„дёӯпјҢжҠҠжҜҸдёӘдҫқиө–йғҪйҖҡзҹҘдёҖйҒҚпјҢе‘ҠиҜүд»–д»¬еә”иҜҘжӣҙж–°дәҶгҖӮ
еңЁgetterдёӯж”¶йӣҶдҫқиө–дёәи°Ғз”ЁеҲ°дәҶиҝҷдёӘж•°жҚ®пјҢе°ұд»ЈиЎЁи°Ғдҫқиө–иҝҷдёӘж•°жҚ®гҖӮ
еңЁsetterдёӯйҖҡзҹҘдҫқиө–жӣҙж–°гҖӮ
жҜҸдёӘж•°жҚ®йғҪжңүдёҖдёӘдҫқиө–з®ЎзҗҶеҷЁDepзұ»пјҡеңЁsrc/core/observer/dep.jsдёӯ

5.3 Watcherи®ўйҳ…иҖ…

и°Ғз”ЁеҲ°дәҶж•°жҚ®пјҢи°Ғе°ұжҳҜдҫқиө–пјҢе°ұдёәи°ҒеҲӣе»әдёҖдёӘWatcherе®һдҫӢпјҢеңЁд№ӢеҗҺж•°жҚ®еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢжҲ‘д»¬дёҚеҺ»зӣҙжҺҘйҖҡзҹҘдҫқиө–жӣҙж–°пјҢиҖҢжҳҜйҖҡзҹҘдҫқиө–еҜ№еә”зҡ„Watcherе®һдҫӢпјҢз”ұWatcherе®һдҫӢеҺ»йҖҡзҹҘзңҹжӯЈзҡ„и§ҶеӣҫгҖӮ

watcherзұ»д»Јз Ғе®һзҺ°йҖ»иҫ‘пјҡ

  1. еҪ“е®һдҫӢеҢ–Watcherзұ»ж—¶пјҢдјҡе…Ҳжү§иЎҢе…¶жһ„йҖ еҮҪж•°гҖӮ
  2. еңЁжһ„йҖ еҮҪж•°дёӯи°ғз”Ёthis.get()ж–№жі•гҖӮ
  3. еңЁget()ж–№жі•дёӯпјҢйҰ–е…ҲйҖҡиҝҮwindow.target=thisжҠҠе®һдҫӢиҮӘиә«иөӢз»ҷдәҶе…ЁеұҖзҡ„дёҖдёӘе”ҜдёҖеҜ№иұЎwindow.targetдёҠпјҢз„¶еҗҺйҖҡиҝҮlet value=this.getter.call(vm, vm)иҺ·еҸ–дёҖдёӢиў«дҫқиө–зҡ„ж•°жҚ®пјҢиҺ·еҸ–иў«дҫқиө–ж•°жҚ®зҡ„зӣ®зҡ„жҳҜи§ҰеҸ‘иҜҘж•°жҚ®дёҠйқўзҡ„getterгҖӮеңЁgetterйҮҢдјҡи°ғз”Ёdep.depend()ж”¶йӣҶдҫқиө–пјҢиҖҢеңЁdep.depend()дёӯеҸ–еҲ°жҢӮиҪҪwindow.targetдёҠзҡ„еҖје№¶е°Ҷе…¶еӯҳе…Ҙдҫқиө–ж•°з»„дёӯпјҢеңЁget()ж–№жі•жңҖеҗҺе°Ҷwindow.targetйҮҠж”ҫжҺүгҖӮ
  4. иҖҢеҪ“ж•°жҚ®еҸҳеҢ–ж—¶пјҢдјҡи§ҰеҸ‘ж•°жҚ®зҡ„setterпјҢеңЁsetterдёӯи°ғз”ЁдәҶdep.notify()ж–№жі•пјҢеңЁdep.notify()ж–№жі•дёӯпјҢйҒҚеҺҶжүҖжңүдҫқиө–(еҚіwatcherе®һдҫӢ)пјҢжү§иЎҢдҫқиө–зҡ„update()ж–№жі•пјҢд№ҹе°ұжҳҜWatcherзұ»дёӯзҡ„update()е®һдҫӢж–№жі•пјҢеңЁupdate()ж–№жі•дёӯи°ғз”Ёж•°жҚ®еҸҳеҢ–зҡ„жӣҙж–°еӣһи°ғеҮҪж•°пјҢд»ҺиҖҢжӣҙж–°и§ҶеӣҫгҖӮ

з®ҖеҚ•жҖ»з»“дёҖдёӢе°ұжҳҜпјҡWatcherе…ҲжҠҠиҮӘе·ұи®ҫзҪ®еҲ°е…ЁеұҖе”ҜдёҖзҡ„жҢҮе®ҡдҪҚзҪ®пјҲwindow.targetпјүпјҢз„¶еҗҺиҜ»еҸ–ж•°жҚ®гҖӮеӣ дёәиҜ»еҸ–дәҶж•°жҚ®пјҢжүҖд»Ҙдјҡи§ҰеҸ‘иҝҷдёӘж•°жҚ®зҡ„getterгҖӮжҺҘзқҖпјҢеңЁgetterдёӯе°ұдјҡд»Һе…ЁеұҖе”ҜдёҖзҡ„йӮЈдёӘдҪҚзҪ®иҜ»еҸ–еҪ“еүҚжӯЈеңЁиҜ»еҸ–ж•°жҚ®зҡ„WatcherпјҢе№¶жҠҠиҝҷдёӘWatcherж”¶йӣҶеҲ°DepдёӯеҺ»гҖӮж”¶йӣҶеҘҪд№ӢеҗҺпјҢеҪ“ж•°жҚ®еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢдјҡеҗ‘Depдёӯзҡ„жҜҸдёӘWatcherеҸ‘йҖҒйҖҡзҹҘгҖӮйҖҡиҝҮиҝҷж ·зҡ„ж–№ејҸпјҢWatcherеҸҜд»Ҙдё»еҠЁеҺ»и®ўйҳ…д»»ж„ҸдёҖдёӘж•°жҚ®зҡ„еҸҳеҢ–гҖӮ

дёҚи¶ід№ӢеӨ„
иҷҪз„¶жҲ‘д»¬йҖҡиҝҮObject.definePropertyж–№жі•е®һзҺ°дәҶеҜ№objectж•°жҚ®зҡ„еҸҜи§ӮжөӢпјҢдҪҶжҳҜд»…иғҪи§ӮжөӢеҲ°objectж•°жҚ®зҡ„еҸ–еҖјеҸҠи®ҫзҪ®еҖјпјҢеҪ“жҲ‘д»¬еҗ‘objectж•°жҚ®йҮҢж·»еҠ дёҖеҜ№ж–°зҡ„key/valueжҲ–еҲ йҷӨдёҖеҜ№е·Іжңүзҡ„key/valueж—¶пјҢе®ғжҳҜж— жі•и§ӮжөӢеҲ°зҡ„пјҢеҜјиҮҙеҪ“жҲ‘д»¬еҜ№objectж•°жҚ®ж·»еҠ жҲ–еҲ йҷӨеҖјж—¶пјҢж— жі•йҖҡзҹҘдҫқиө–пјҢж— жі•й©ұеҠЁи§ҶеӣҫиҝӣиЎҢе“Қеә”ејҸжӣҙж–°гҖӮ
еҪ“з„¶пјҢVueд№ҹжіЁж„ҸеҲ°дәҶиҝҷдёҖзӮ№пјҢдёәдәҶи§ЈеҶіиҝҷдёҖй—®йўҳпјҢVueеўһеҠ дәҶдёӨдёӘе…ЁеұҖAPI:Vue.setе’ҢVue.delete

Compilerдё»иҰҒзҡ„дҪңз”ЁжҳҜжҠҠVueз»‘е®ҡзҡ„DOMиҠӮзӮ№пјҲд№ҹе°ұelж Үзӯҫз»‘е®ҡзҡ„idпјүйҒҚеҺҶиҜҘиҠӮзӮ№зҡ„жүҖжңүеӯҗиҠӮзӮ№пјҢжүҫеҮәе…¶дёӯжүҖжңүзҡ„v-жҢҮд»Өе’Ң {{ }} гҖӮ

  1. еҰӮжһңеӯҗиҠӮзӮ№еҗ«жңүv-жҢҮд»ӨпјҢеҚіжҳҜе…ғзҙ иҠӮзӮ№пјҢеҲҷеҜ№иҝҷдёӘе…ғзҙ ж·»еҠ зӣ‘еҗ¬дәӢд»¶гҖӮеҰӮжһңжҳҜv-onпјҢеҲҷnode.addEventListener(вҖҳclickвҖҷ)пјҢеҰӮжһңжҳҜv-modelпјҢеҲҷnode-addEventListener(вҖҳinputвҖҷ)гҖӮжҺҘзқҖеҲқе§ӢеҢ–жЁЎжқҝе…ғзҙ пјҢеҲӣе»әдёҖдёӘWatcherз»‘е®ҡиҝҷдёӘе…ғзҙ иҠӮзӮ№гҖӮ
    йҖҡиҝҮhasAttributeеҺ»еҲӨж–ӯжҹҗдёӘиҠӮзӮ№жҳҜеҗҰеӯҳеңЁжҲ‘д»¬иҮӘе®ҡд№үзҡ„жҢҮд»ӨпјҢйҖҡиҝҮgetAttributeеҺ»иҺ·еҸ–иҝҷдёӘжҢҮд»ӨжүҖз»‘е®ҡзҡ„dataеҜ№иұЎе…·дҪ“зҡ„еұһжҖ§пјҲд№ҹе°ұжҳҜйў‘йҒ“пјүгҖӮ

v-modelзҡ„е…·дҪ“дҪңз”ЁжҳҜз»‘е®ҡinputжҲ–textareaж Үзӯҫзҡ„иҫ“е…ҘдәӢд»¶пјҢеҪ“иҫ“е…Ҙж”№еҸҳж—¶пјҢж”№еҸҳdataеҜ№иұЎдёӯе…·дҪ“еұһжҖ§зҡ„еҖјгҖӮ

v-bindе°ұжҳҜе“Қеә”и®ўйҳ…

  1. еҰӮжһңеӯҗиҠӮзӮ№жҳҜж–Үжң¬иҠӮзӮ№пјҢеҚі {{ data }} пјҢеҲҷз”ЁжӯЈеҲҷиЎЁиҫҫејҸеҸ–еҮә {{ data }} дёӯзҡ„dataпјҢз„¶еҗҺvar initText = this.vm[exp]пјҢз”ЁinitTextеҺ»д»Јжӣҝе…¶дёӯзҡ„dataгҖӮ
    и§Јжһҗ {{ }} и·ҹv-bindзҡ„жҖқи·ҜдёҖж ·пјҢеҢәеҲ«еңЁдәҺеҸҢжӢ¬еҸ·иҜӯжі•жҳҜеӯҳеңЁдәҺж–Үжң¬иҠӮзӮ№дёӯзҡ„пјҢжҲ‘д»¬ж— жі•йҖҡиҝҮgetAttributeеҺ»зӣҙжҺҘиҺ·еҸ–пјҢжүҖд»ҘжҲ‘д»¬иҰҒи§ЈжһҗеӯҳеңЁеҸҢжӢ¬еҸ·иҜӯжі•зҡ„ж–Үжң¬иҠӮзӮ№пјҡ

  3. з®ҖеҢ–дәҶи°ғиҜ•жӯҘйӘӨ
  4. жҸҗеҚҮдәҶйЎ№зӣ®зҡ„еҸҜз»ҙжҠӨжҖ§

  • v-htmlпјҡv-textе’ҢжҸ’еҖјиЎЁиҫҫејҸеҸӘиғҪжёІжҹ“зәҜж–Үжң¬еҶ…е®№пјҢеҰӮжһңиҰҒжҠҠеҢ…еҗ«HTMLж Үзӯҫзҡ„еӯ—з¬ҰдёІжёІжҹ“дёәйЎөйқўзҡ„HTMLе…ғзҙ пјҢеҲҷйңҖиҰҒдҪҝз”Ёv-htmlжҢҮд»ӨгҖӮ дҪҝз”ЁеҰӮдёҠж–№жі•ж—¶пјҢusernameгҖҒdiscriptionзӯүеҖјйғҪиҰҒеҶҷеңЁdataйҮҢпјҢе…¶дёӯelиҰҒжҢҮеҗ‘ж•ҙдёӘеӨ§зӣ’еӯҗпјҢи°ғз”ЁdataйҮҢзҡ„е…ғзҙ йғҪиҰҒж”ҫеҲ°иҝҷдёӘеӨ§зӣ’еӯҗйҮҢжүҚиғҪз”ҹж•ҲгҖӮ
