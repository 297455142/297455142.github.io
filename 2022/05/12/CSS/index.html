CSS

零、网页布局准则

一、多个块级元素纵向排列找标准流，多个块级元素横向排列找浮动流。
二、先设置盒子大小，再设置盒子位置。

一般策略：先用标准流的父元素排列上下位置，再用内部子元素采取浮动排列找左右位置。

一、CSS常用属性

一、文字相关
  • font-family：定义文本的字体，可以多个字体，每个字体用逗号分隔，靠前的字体先用。
  • font-size：定义字体的大小，单位为px。
  • font-weight：设置文字粗细，normal正常，bold粗体。可以直接写数字，400为normal，700为bold。
  • font-style：设置字体样式，normal为正常。italic斜体，inherit继承父元素样式。
  • font：每个具体属性用空格隔开，顺序不能更换：font-style、font-weight、font-size/line-height、font-family。必须有font-size和font-family。
  • color：设置文字颜色。可以直接英文单词，或者#后跟十六进制树，或者rgb(a,b,c)
  • text-align：设置文本内容的水平对齐方式center、left、right。行内元素或行内块元素水平居中给其父元素添加text-align:center。
  • text-decoration：给文本添加下划线（underline）、上划线（overline）、删除线（line-through）、默认无线（none）。一般用于去除链接的下划线。
  • text-indent：用于指定文本第一行的缩进，通常是将段落首行进行缩进。单位为px或em（1个文字大小）。
  • line-height：用于设置行高。文字垂直居中时直接设置行高和盒子高一样。行高小于盒子高度文字会偏上，行高大于盒子高度文字会偏下。
  • text-shadow：文本阴影。h-shadow必须，水平阴影的位置，允许负值。v-shadow必须，垂直阴影的位置。blur可选，模糊距离，表示阴影的深浅。color可选，阴影颜色。
    二、盒子背景相关
  • background-color：背景颜色，transparent默认透明色。
  • background-image：背景图片，便于控制位置、logo或者大图片。none或者url(url)。
  • background-repeat：对背景图片进行平铺repeat、不平铺no-repeat、x轴方向平铺repeat-x、y轴方向平铺repeat-y。默认是平铺的。
  • background-position：改变图片在背景中的位置。x和y代表坐标，可以使用方位名词或精确单位。方位名词和顺序没有关系，只写一个方位另一个默认居中对其：left、right、top、center、bottom。精确单位先x后y。可以使用混合单位，第一个一定是x，第二个一定是y。
  • background-attachment：设置背景图像固定或者随着页面滚动。scroll滚动（默认），fixed固定。
  • background：背景颜色、背景图片地址、背景平铺、背景图像滚动、背景图片位置。
  • background:rgba(0,0,0,0)：前面三个是rgb，最后是透明(0~1)。只有背景色半透明。
    三、盒子模型
  • border：边框UC逆袭，样式，颜色。
  • border-width：边框粗细，单位px。边框占盒子以外宽度，边框会额外增加盒子的实际大小，测量时要么不量边框，要么高宽减去边框粗细。
  • border-style：边框样式，默认none，solid实线边框，dashed虚线边框，dotted点线边框。
  • border-color：边框颜色。
  • border-collapse：collapse合并两个盒子的边框。
  • border-radius：用于设置元素的外边框圆角，数值length是圆的半径，单位可以是px或者%。左上角、右上角、右下角、左下角。若想使用椭圆角，每个值为apx/bpx。
  • border-top-left-radius、border-top-right-radius、border-bottom-left-radius、border-bottom-right-radius
  • padding：内边距，单位px。内边距会撑大盒子原来宽高，如果盒子本身没有指定width/height属性，则此时padding不会撑开盒子大小。
  • padding-left、padding-right、padding-top、padding-bottom：左右上下边框。
  • margin：外边距，控制盒子之间的距离，用法同上。
  • box-shadow：为盒子添加阴影。h-shadow必须，水平阴影的位置，允许负值。v-shadow必须，垂直阴影的位置，允许负值。blur可选，模糊距离，表示阴影的深浅。spread可选，阴影的尺寸。inset可选，将外部阴影改为内部阴影，默认outset。阴影不占空间。
    四、杂项
  • list-style：none。去除li前面的小圆点。
  • float：浮动。none不浮动，默认值。left元素左浮动，right元素右浮动。

三、CSS盒模型

两种盒模型，box-sizing属性：content-box | border-box | inherit

  • content-box 默认值：标准盒模型。边框border和内边距padding的值不包含在width之内。对象的实际宽度等于设置的width值和border、padding之和。
  • border-box IE盒模型。边框border和内边距padding的值是包含在width之内的。对象的实际宽度就等于设置的width值。

四、块级元素和行级元素（内联元素）

  1. 块级元素：每个块级元素独占一行，元素的高度、宽度、行高和边距可设置，元素的宽度不设置的话默认为父元素的宽度（父元素宽的100%），里面可以放其他元素，文字类块元素中不允许再放其他元素。包括：div、p、ul、li、h、form、hr、ol、dl、dt、dd、address、article、figure、audio、video、section、table、canvas、header、footer。
  2. 行级元素：每个行内元素可以和别的行内元素共享一行，相邻的行内元素会排列到同一行内，直到一行排不下了，才会换行。行内元素的高度、宽度、行高及顶部和底部边距不可设置。元素的宽度就是它包含的文字或图片的宽度，不可改变。包括span、a、br、img、input、textare、select、sup、sub、em、del
  3. 行内块元素：一行上可以有多个，默认为自己的宽，高度宽度内外边距都可以控制。包括img、input、td。
  4. 区别：
    ①排列方式：块级元素会独占一行，内联元素和内联块元素会在一行内显示。
    ②宽高边距设置：块级元素和内联块元素可以设置宽高，内联元素设置无效。
    ③默认宽度：块级元素width默认为100%，内联元素则是根据自身的内容或子元素来决定其宽度。
  5. 转换
  • display: block; 把行内元素转换为块元素。
  • display：inline; 把块元素转换为行内元素。
  • display: inline-block; 把元素转换为行内块元素。

五、CSS的单位：

  • px：像素，相对长度单位，相对于显示器屏幕分辨率而言的。可以设置绝对字体大小。1px在高清屏幕下可能占用2个甚至3个物理像素。
  • em：相对长度单位，相对于当前对象内文本的一个字体的尺寸。根据其父元素的字体大小font-size来设置。
  • rem：相对单位，相对于根元素（一般为html）的font-size来计算，适用于响应式页面。通过它既可以做到只修改根元素就成比例地调整所有字体的大小，又可以避免字体大小逐层复合的连锁反应。
  • %：相对于父元素的长度高度。position:fixed时，无论当前元素仿真哪里，将相对于窗口宽度。position:absolute时，将相对于其相对的元素宽度（递归父元素直到第一个设置了position的元素）
  • vh/vw：相对于视窗Viewport的高度和宽度，1vh等于1/100的视窗高度，1vw等于1/100的视窗宽度。视窗是指浏览器内部的课时区域大小，即window.innerWidth/window.innerHeight大小，不包含任务栏标题栏以及底部工具栏的浏览器区域大小。
  • vmin/vmax：vmin取视窗高度和宽度的最小值作为基准。vmax取视窗高度和宽度的最大值作为基准。使用场景是保持字体在横屏和竖屏状态下显示的字体大小一致。

七、伪类与伪元素

CSS引入伪类和伪元素概念是为了格式化文档树之外的信息，也就是说伪类和伪元素是用来修饰不在文档树中的部分。比如，一句话中的一个字母，或者是列表中的第一个元素。

伪类

伪类用于定义元素的特殊状态。如：

  • a:link 选择所有未被访问的链接。
  • a:visited 选择所有已被访问的链接。
  • a:hover 选择鼠标指针位于其上的链接。
  • a:active 选择活动链接（鼠标按下未抬起的链接）
  • :focus 选取获得焦点的表单元素，一般input类表单元素才能获取。
    伪类用于当已有元素处于某个状态时，为其添加对应的样式，这个样式是根据用户的行为而动态变化的。
    伪元素
    伪元素用于将特殊的效果添加到某些选择器，伪元素代表了某个元素的子元素，这个元素虽然逻辑上存在，但却并不实际存在于文档树中。
    伪元素用于设置元素指定部分的样式，如：

八、CSS选择器与权重

1、!important：加在样式属性值后，权重值为无穷大。如color:green!important。
2、行内样式style=””：权重1000，直接写在标签内。
3、ID选择器：权重100，如#content。
4、类、伪类和属性选择器，如content、:hover，权重为10。
5、标签选择器和伪元素选择器，如div、p、::before权重值为10。
6、通配符选择器（*）、子选择器（>）、相邻选择器（+）、同胞选择器（~），权重为0。
7、后代选择器：如div .title {color: red}。前一个选择器跟后一个选择器。
8、子元素选择器：只在儿子层找，如div>p {color: red}
9、并集选择器：同时匹配所有选择元素，div,p,h1 {color: red}
10、毗邻选择器：紧挨着相邻的，只找下面的不找上面的，只找一个。如.c2+p {color: red}
11、兄弟选择器：同一级之后的所有，如.c2~p {color: red}

九、实现扇形、半圆、圆形、椭圆

扇形

一个角是圆角，其它三个角都是直角。圆角的值为宽和高一样的值，其它三个角的值为0。border-radius的四个值顺序是：左上角、右上角、右下角、左下角。

半圆

相邻的两个角的半径一样。

圆

正方形里四个角的半径都是宽高的一半。

椭圆

盒子长宽不同直接用相同百分比或apx/bpx。

十、网页布局的三种方式

  • 标准文档流：标签按照默认规定好的方式进行排列。从上到下，从左到右，盒子模型尺寸等于边框+外边距+内边距+盒子中间内容尺寸。
  • 浮动流：可以改变元素标签默认的排列方式。
  • 绝对定位流：包括静态定位、相对对位、绝对定位。

十一、浮动float、高度塌陷和去除浮动

一、浮动float
  • 浮动属性用于创建浮动框，将其按照指定方向移动，直到左边缘或者右边缘触及包含块或另一个浮动框的边缘（直接贴到一起）。
  • float：浮动。none不浮动，默认值。left元素左浮动，right元素右浮动。
    二、浮动特性
  • 浮动元素会脱离标准流的控制，移动到指定位置，浮动的盒子不再保留原来的位置，浮动只会压住后面的标准流，不会压住前面的。
  • 浮动的元素会一行内显示并且元素顶端对齐，如果装不下了会自动换行。
  • 浮动的元素会具有行内块元素的特性。如果元素有了浮动，不需要转换直接可以给高度和宽度。如果元素没有设置宽度，添加浮动后，它的大小根据内容来决定。
    三、高度塌陷
    一个元素设置了浮动之后，会影响其兄弟元素的位置和父级盒子的高度。父级盒子在很多情况下，不方便给高度，但是盒子浮动不占有位置，不会撑起父级盒子的高度，最后导致父级盒子高度为0，会影响后续的标准流盒子。
    1、如果兄弟元素是块级元素，会无视这个浮动的边框，使自身尽可能与这个浮动元素处于同一行，导致被浮动元素覆盖。除非这些div设置了宽度，并且父元素的宽度不足以包含它们，导致强制换行。
    2、若是内联元素，则会尽可能围绕浮动元素。如浮动元素不会压住后续盒子里的文字，文字会围绕浮动元素显示。
    四、清除浮动
  • 额外标签法（隔离法）：在浮动元素末尾添加一个空的标签（必须为块级元素），并将该标签设置为clear:both。
  • 父级元素添加overflow属性：属性值设定为hidden、auto或scroll。用在同一个容器内形成BFC独立容器避免浮动造成的元素塌陷。
